
# 微信自动化抢红包助手

> 微信最新版本8.0.68已测试通过，其它版本未知

## 原理

> 通过屏幕元素捕捉的方式抢红包，所以抢红包时需要：1.亮屏。2.屏幕停留在微信群聊页面。3.抢红包期间不能操作屏幕。

## 使用方法

1. 下载安装`AIGAME.apk`，AIGAME官网链接：[http://ermxx.com/api/act.html](http://ermxx.com/api/act.html)

2. 打开 `AIGAME App`，新建项目

3. 将本项目的`icon.png`, `StepBox.js`, `main.js`, `main.xml`, `setting.xml`替换掉你新建项目的文件即可

4. 运行`main.js`文件
